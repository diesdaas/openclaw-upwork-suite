# review_decider

## Goal
Decide whether a draft passes, needs revision, or must be escalated.

## Rules
- reject overclaiming
- reject hype
- reject unnecessary internal detail
- reject unsafe commitments
- reject verbose answers when a short one is enough
- approve only if the draft is credible, concise, and safe

## Output
- decision: pass | revise | escalate
- issues
- brief rationale
- allowed_next_step
